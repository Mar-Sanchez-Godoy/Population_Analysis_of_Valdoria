# Valdoria — Fictional City Economic & Demographic Simulation

## Project Description

This repository documents the complete process of demographic, economic, and logistical analysis of Valdoria, a fictional city created for the novel *Hambruna*.

The project explores how AI-assisted analytical workflows can be used to generate structured models, hypotheses, calculations, and urban simulations for fictional environments.

It demonstrates how tools like ChatGPT can support analytical reasoning similar to workflows used in Business Intelligence and data analysis.

The goal is twofold:

- Build a coherent internal economic system for a fictional city
- Showcase data modeling, simulation, and analytical thinking

---

## Project Scope

This project includes:

- Synthetic data generation  
- Economic and demographic modeling  
- Population distribution simulations  
- Urban logistics analysis  
- Mathematical and statistical reasoning  
- Visualizations and structured datasets  

---

## Narrative Context

Valdoria is a port city located in a fertile valley, shaped by trade routes, migration flows, and resource scarcity.

Within the narrative universe of the novel *Hambruna*, the city faces growing economic pressure, demographic imbalance, and logistical challenges related to food distribution.

Rather than treating the setting purely as fiction, this project constructs a realistic internal model of how such a city could function.

---

## Methodology

The analytical workflow was built through iterative prompting and model refinement using ChatGPT.

The process followed several stages:

1. Base demographic modeling  
2. Economic structure estimation  
3. Urban logistics modeling  
4. Food supply chain simulation  
5. Mathematical validation  
6. Dataset generation  

Each stage progressively refined the previous assumptions.

---

## Prompts Used

### Prompt 01 — Base Demographic Data Generation

“How many houses, people, crops, and castle storage rooms would you estimate are needed to feed the city?”

Goal:

- Estimate population scale  
- Define housing distribution  
- Establish initial food production assumptions  

---

### Prompt 02 — Economic Analysis

“Give me figures and organize everything into one or several tables so it can be analyzed graphically.”

Goal:

- Define production sectors  
- Simulate economic activity  
- Generate structured datasets  

---

### Prompt 03 — Urban Logistics

“Model the distribution of goods and transportation routes…”

Goal:

- Simulate internal trade flows  
- Model transportation and storage infrastructure  

---

### Prompt 04 — Economic Flow Map

Prompt:

“What would the internal economic flow of Valdoria look like?”

Result:

Creation of a city-scale flow model including:

- Food production  
- Labor allocation  
- Distribution networks  
- Resource bottlenecks  

---

### Prompt 05 — Explainability

Prompt:

“Since this is a fictional city, explain:
- Which parts are invented
- Which parts are based on real models
- Which assumptions were used
- Which limitations the model has”

Goal:

Provide transparency about:

- Synthetic assumptions  
- Real-world economic analogues  
- Model constraints  

---

### Prompt 06 — Technical and Mathematical Analysis

Prompt:

“Technical and Mathematical Analysis.
As a BI Analyst, this is where I shine.”

Includes:

- Formulas used  
- Statistical models  
- Mathematical assumptions  
- Tables and charts  
- Step-by-step simulations  

---

### Prompt 07 — CSV Dataset Generation

Prompt:

“Using the economic flow map and each model, generate CSV tables.”

Goal:

Produce structured datasets suitable for analysis or visualization tools.
